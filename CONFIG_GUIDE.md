# 导航菜单配置说明

## 配置文件位置

`public/config.js` - 所有导航菜单的配置都在这个文件中

## 快速开始

### 1. 隐藏某个顶级分类（如：不显示"动漫"）

编辑 `public/config.js`，在 `hiddenParentTypes` 数组中添加分类 ID：

```javascript
hiddenParentTypes: [
  4, // 不显示 动漫
],
```

### 2. 隐藏某个无下拉菜单的分类（如：不显示"福利"）

编辑 `public/config.js`，在 `hiddenNoDropdownTypes` 数组中添加分类 ID：

```javascript
hiddenNoDropdownTypes: [
  19, // 不显示 福利
],
```

### 3. 隐藏某个子分类（如：不显示"伦理片"）

编辑 `public/config.js`，在 `hiddenSubCategories` 数组中添加分类 ID：

```javascript
hiddenSubCategories: [
  61, // 不显示 伦理片
],
```

### 4. 完全禁用某个子分类（如：不显示电影分类下的"预告片"）

编辑 `public/config.js`，从 `categoryChildren` 中移除对应的 ID：

```javascript
categoryChildren: {
  1: [5, 6, 7, 8, 9, 10, 11, 20, 41], // 移除了 61, 92, 93
  // ...
}
```

## 配置说明

### parentTypes
有下拉菜单的父分类 ID 列表
- `1` = 电影
- `2` = 连续剧
- `3` = 综艺
- `4` = 动漫

### noDropdownTypes
没有下拉菜单，直接点击跳转的分类
- `19` = 福利
- `83` = 短剧大全
- `94` = 体育

### categoryChildren
每个父分类对应的子分类 ID 列表

**电影 (1) 的子分类：**
- `5` = 动作片
- `6` = 喜剧片
- `7` = 爱情片
- `8` = 科幻片
- `9` = 恐怖片
- `10` = 剧情片
- `11` = 战争片
- `20` = 记录片
- `41` = 动画片
- `61` = 伦理片
- `92` = 预告片
- `93` = 预告片

**连续剧 (2) 的子分类：**
- `12` = 国产剧
- `13` = 台湾剧
- `14` = 韩国剧
- `15` = 欧美剧
- `16` = 香港剧
- `17` = 泰国剧
- `18` = 日本剧
- `54` = 海外剧

**综艺 (3) 的子分类：**
- `62` = 大陆综艺
- `63` = 港台综艺
- `64` = 日韩综艺
- `65` = 欧美综艺

**动漫 (4) 的子分类：**
- `66` = 国产动漫
- `67` = 日韩动漫
- `68` = 欧美动漫
- `69` = 港台动漫
- `70` = 海外动漫

### hiddenSubCategories
不想显示的子分类 ID 列表（从下拉菜单中隐藏）

### hiddenParentTypes
不想显示的父分类 ID 列表（完全隐藏某个顶级分类）

### hiddenNoDropdownTypes
不想显示的无下拉菜单分类

### categoryNames
自定义分类显示名称（可选）

```javascript
categoryNames: {
  1: '电影',
  6: '喜剧', // 将"喜剧片"改为"喜剧"
}
```

## 修改配置后的操作

修改 `public/config.js` 后，需要重新构建：

```bash
npm run build
```

然后重新部署到服务器。

## 示例配置

### 示例 1：只显示电影和连续剧，隐藏伦理片

```javascript
export const navConfig = {
  parentTypes: [1, 2], // 只显示电影和连续剧
  noDropdownTypes: [], // 不显示独立分类
  categoryChildren: {
    1: [5, 6, 7, 8, 9, 10, 11, 20, 41], // 电影分类，移除伦理片和预告片
    2: [12, 13, 14, 15, 16, 17, 18, 54] // 连续剧分类
  },
  hiddenSubCategories: [],
  hiddenParentTypes: [3, 4], // 隐藏综艺和动漫
  hiddenNoDropdownTypes: [19, 83, 94] // 隐藏福利、短剧大全、体育
};
```

### 示例 2：显示所有分类，但隐藏伦理片和预告片

```javascript
export const navConfig = {
  parentTypes: [1, 2, 3, 4],
  noDropdownTypes: [19, 83, 94],
  categoryChildren: {
    1: [5, 6, 7, 8, 9, 10, 11, 20, 41, 61, 92, 93],
    2: [12, 13, 14, 15, 16, 17, 18, 54],
    3: [62, 63, 64, 65],
    4: [66, 67, 68, 69, 70]
  },
  hiddenSubCategories: [61, 92, 93], // 隐藏伦理片和预告片
  hiddenParentTypes: [],
  hiddenNoDropdownTypes: []
};
```

### 示例 3：极简模式，只显示主要分类

```javascript
export const navConfig = {
  parentTypes: [1, 2, 3, 4],
  noDropdownTypes: [],
  categoryChildren: {
    1: [5, 6, 7, 8, 10], // 只显示主要电影类型
    2: [12, 14, 15, 18], // 只显示主要剧集类型
    3: [62, 64, 65], // 只显示主要综艺类型
    4: [66, 67, 68] // 只显示主要动漫类型
  },
  hiddenSubCategories: [],
  hiddenParentTypes: [],
  hiddenNoDropdownTypes: [19, 83, 94]
};
```

## 注意事项

1. **修改配置后必须重新构建**：`npm run build`
2. **生产环境**：构建后的文件在 `dist/` 目录
3. **开发环境**：修改配置后刷新浏览器即可看到效果
4. **备份配置**：修改前建议备份 `config.js` 文件
